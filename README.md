# Sentiment Analysis of Customer Reviews

## Project Overview  
This project focuses on analyzing customer reviews using **Natural Language Processing (NLP)** techniques to determine the sentiment of reviews (positive, negative, or neutral). The application is built using **Streamlit, Pandas, and Matplotlib**, providing an interactive dashboard for visualizing sentiment trends.  

## Features  
✔️ **Upload & Analyze Customer Reviews** – Supports CSV file uploads for bulk analysis.  
✔️ **Sentiment Classification** – Uses basic text processing to classify reviews as positive, negative, or neutral.  
✔️ **Interactive Dashboard** – Displays insights through graphs and charts.  
✔️ **Data Filtering & Visualization** – Allows filtering based on sentiment type.  
✔️ **Future Enhancement** – Will integrate **Machine Learning models** for more accurate sentiment predictions.  

## Technology Stack  
- **Frontend:** Streamlit (for interactive UI)  
- **Backend:** Python  
- **Libraries Used:** Pandas, Matplotlib, NLTK (for NLP processing)  
- **Dataset:** Amazon Reviews Dataset (CSV format)
